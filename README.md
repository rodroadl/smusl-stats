# SMUSL-STATS
The statistics for the clubs playing in Southern Maine United Soccer League (SMUSL) from 2021 to today.
The statistics includes:
- Elo rating
- Match odd (Win/Draw/Lose)
- Biggest upset
- Most Unexpected
- and to be added more..
